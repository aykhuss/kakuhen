
kakuhen documentation
=====================

A header-only C++ multidimensional Monte Carlo integration library.

Key features
------------

* Header-only, C++20, no build system required to consume.
* Multiple integrators (Plain, Vegas, Basin).
* Thread-friendly histogramming utilities.

Getting started
---------------

Start with the :doc:`Quickstart <guides/quickstart>` to run your first integration,
then see :doc:`Installation <guides/installation>` for recommended CMake usage.
